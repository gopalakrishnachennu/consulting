Control how new jobs flow through the review process before going live to consultants.

{{ form.require_pool_staging }}

When enabled, all new jobs (created manually or via bulk upload) land in the Job Pool for review before being visible to consultants. Disable this only if you want backward-compatible behaviour where jobs go live immediately.

ON: Create → Pool → Review → Open
OFF: Create → Open immediately (no vetting)
